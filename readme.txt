Track page views and trigger targeted automations based on Mailchimp campaign engagement.

== Description ==

Track and analyze page view events when subscribers click links in your
Mailchimp email campaigns. Leverage the Mailchimp API to collect website
visitor behavior and trigger targeted automations based on user
engagement. Perfect for improving conversion rates and creating
personalized follow-up campaigns.
